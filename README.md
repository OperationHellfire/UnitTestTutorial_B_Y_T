# Usage Guide
To test the functionality of these two tasks, you may either just copy the .cs files from their respective files in a new build

or

Use the provided .sln files for Visual Studio Community.

To run the tests, please run Program.cs found in their respective files.

ATTENTION: There is a possibility that NUnit won't be installed for both scenarios on a new computer so in the case of that event, please make sure NUnit is installed via NuGet.
